Lanciando il file "main.py" si apre l'interfaccia della rubrica telefonica con tutte le operazioni possibili.
C'è la possibilità di chiamare un helper con la descrizione dettagliata del funzionamento della rubrica e di tutte le operazioni.

La rubrica è inizializzata con 3 contatti.
Nella cartella c'è un file json per poter testare l'inserimento dei contatti da file; questo file contiene 3 contatti, di cui soltanto uno è corretto per l'inserimento.
